Example correlation (pair) plot for retrieved phases. Note in this case the reference phase (top left panel) has been set to 0 to illustrate general patterns in the retrieved parameter sets. In general, the curvature indicates how well-defined a given parameter is, whilst V-patterns indicate cases with \(\pm\phi\) pairs. The bottom row and final column on the right show patterns of the parameters with respect to \(\bar{\chi}^2\).
