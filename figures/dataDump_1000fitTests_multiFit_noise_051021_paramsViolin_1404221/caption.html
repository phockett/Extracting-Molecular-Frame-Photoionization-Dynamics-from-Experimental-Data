Statistics for the 349 best fit results (\(\chi^2 < 2.3 \times 10^{-4}\)). Plots show magnitudes (top) and phase (bottom) parameters, labelled by \((l,m)\), with spread proportional to the number of results and colour-mapped by \(\chi^2\) . Note the clustering by \(\chi^2\), where each cluster constitutes a viable set of matrix elements, and the lack of variation for the \(|1,-1\rangle\) phase, which was fi as a reference
